## 🚀 Featured Projects

### 📅 Event Planner & RSVP Management

A full-stack event management application where authenticated users can create events, generate public invitation links and track guest responses.

**Key features:**

* Secure authentication and protected routes
* Event creation and management
* Public token-based invitation links
* Going, Maybe and Not Going RSVP tracking
* User-specific dashboard
* Dynamic updates using Next.js Server Actions

**Technologies:** Next.js, TypeScript, PostgreSQL, Prisma, Neon Auth, Tailwind CSS and shadcn/ui

---

### 💼 Job Hunt Tracker

A full-stack Kanban application that helps users organize job applications across different recruitment stages.

**Key features:**

* Kanban-based job application workflow
* Boards, columns and application cards
* Secure authentication with Better Auth
* Protected routes and ownership validation
* Create, update, move and delete applications
* Server-side rendering and cache revalidation

**Technologies:** Next.js, TypeScript, MongoDB, Mongoose, Better Auth, Server Actions and Tailwind CSS

---

### 🛒 BellHopt

A customer-facing online grocery and food delivery platform designed to provide a smooth e-commerce shopping experience.

**Key features:**

* Product and category browsing
* Shopping cart management
* User authentication
* Order placement
* Checkout workflow
* Responsive interface

**Technologies:** React, Node.js and MongoDB
